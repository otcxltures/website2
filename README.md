# Gadgets by Kihoro

## Project Description
This is a simple website that showcases different gadgets such as phones, laptops, and accessories.  
It is built using HTML and CSS.

## Pages Included
- Home page (index.html)
- Phones page (phones.html)
- Laptops page (laptop.html)
- Accessories page (Accessories.html)
- Cart page (cart.html)

## Technologies Used
- HTML
- CSS

## Features
- Navigation bar for easy movement between pages
- Product images and descriptions
- Simple and clean layout
- Styled using external CSS (style.css)
